# World Happiness Analysis

## 项目简介
这是一个使用Tableau进行的世界幸福度数据分析项目，通过可视化展示全球各国的幸福指数及其影响因素。

## 数据来源
World Happiness Report - 全球幸福度报告数据

## 分析内容
- 各国幸福指数排名
- 影响幸福度的关键因素分析
- 地区幸福度对比
- 历年幸福度趋势变化

## 工具
- Tableau Desktop

## 如何查看
下载 `.twbx` 文件后，使用 Tableau Desktop 或 Tableau Reader 打开查看。

## 在线预览
可在 [Tableau Public](https://public.tableau.com/) 上传后在线查看。
